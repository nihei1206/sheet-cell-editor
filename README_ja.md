# Sheet Cell Editor

*[English README is here](README.md)*

Google スプレッドシートのセルで長文を快適に入力・編集するためのサイドパネルエディタ（Chrome拡張機能）です。

## 機能
- **サイドパネル連携:** Google スプレッドシートの横にChromeのサイドパネルとして便利に展開されます。
- **簡単な読み込みと保存:** 選択したセルの内容をすぐにエディタに読み込み、編集後はワンクリックでセルに書き戻すことができます。
- **同期された編集機能:** 数式バーとシームレスに連携してテキストの読み書きを行うため、Google スプレッドシートのアクティブセルと確実に互換性を保ちます。

## インストール方法
現在、この拡張機能は開発者モードにてパッケージ化されていない拡張機能としてインストールできます。

1. このリポジトリをクローンするか、ダウンロードします。
2. Chromeを開き、`chrome://extensions/` にアクセスします。
3. 右上のトグルスイッチを使って **デベロッパー モード** を有効にします。
4. **パッケージ化されていない拡張機能を読み込む** をクリックし、拡張機能のファイルが含まれるディレクトリを選択します。

## 使い方
1. 編集権限のあるGoogle スプレッドシートを開きます。
2. Chromeのツールバーにある拡張機能アイコンをクリックします。サイドパネルに **Sheet Cell Editor** が開きます。
3. スプレッドシートでセルを選択します。
4. 選択したセルのテキストが自動的にエディタに読み込まれます。
5. 広いテキストエリアで編集を行います。
6. **Save to cell（セルへ書込）** をクリックして、変更をアクティブなセルに書き戻します。

## 技術仕様
この拡張機能はChrome Extension Manifest V3を使用し、Google スプレッドシートと連携するために `sidePanel`、`activeTab`、`scripting` 権限を利用しています。

- **`manifest.json`**: 拡張機能の設定ファイル。
- **`background.js`**: コンテンツスクリプトとサイドパネル間のメッセージを管理するService Worker。
- **`content.js`**: 数式バー（`#t-formula-bar-input`）などのDOMとやり取りするために、Google スプレッドシートに挿入されるコンテンツスクリプト。
- **`sidepanel.html` / `sidepanel.js`**: エディタサイドパネルのUIおよびロジック。

## トラブルシューティング
- **セルに書き込めない:** 「Save to cell」をクリックする前に、セルが選択され、入力可能な状態になっていることを確認してください。この拡張機能は数式バーを操作することで機能します。
